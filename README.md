# PATTERN-SOLVING-ALGORITHMS


This repository contains different star pattern programs written in Python using nested loops.
And Each Problem Has Detailed Explanations For Better UnderStanding

The purpose of this project is to understand how patterns are created by controlling:
- Rows and columns
- Spaces and stars
- Increasing and decreasing loop patterns
- Nested loop logic

## Patterns Included

### 1. Normal Star Pattern
A simple increasing triangle pattern where the number of stars increases with each row.

---

### 2. Inverse Star Pattern
A decreasing triangle pattern where the number of stars reduces after every row.


---

### 3. Mirror Star Pattern
A right-aligned triangle pattern created by adding spaces before the stars.


---

### 4. Inverse Mirror Star Pattern
A right-aligned inverted triangle pattern combining decreasing stars with increasing spaces.

---

### 5. Butterfly Pattern
A combination of two triangle patterns forming a butterfly shape using stars and spaces.


---

## Concepts Practiced

- Python `for` loops
- Nested loops
- Range functions
- Printing patterns using spaces and characters
- Understanding row-wise pattern logic
